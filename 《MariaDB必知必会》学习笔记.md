## 基本语句

选择数据库
```sql
USE my_database;
```

返回可访问数据库的列表
```sql
SHOW DATABASES;
```

返回当前选择的数据库中可用表的列表信息
```sql
SHOW TABLES;
```

显示表结构
```sql
DESCRISE my_table;
```

显示表结构，同上
```sql
SHOW COLOMNS FORM my_table;
```
